TypeCHub

TypeCHub is a custom USB hub PCB designed in EasyEDA. The goal of this project was to learn more about USB hardware,and PCB Design.

"TypeCHub PCB" (PCB-Top.png)

it has

- USB hub controller
- USB-C ports
